# 🔐 Projeto Login – Gabriela Caluz

Este projeto representa uma tela de **login estilizada com HTML e CSS**, desenvolvida como parte do meu aprendizado em desenvolvimento web.

É um exemplo prático de como estruturar e organizar formulários visuais, com foco em clareza, usabilidade e estética.

---

## ✨ Funcionalidades

- Interface de login simples e elegante
- Campos para e-mail e senha
- Layout centralizado
- Estilização com foco em responsividade e contraste

> ⚠️ Este projeto é visual e estático, não possui validação ou backend (JavaScript será adicionado em etapas futuras de aprendizado)

---

## 🧰 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox (para centralização e layout)
- Boas práticas de acessibilidade
